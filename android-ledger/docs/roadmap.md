# Android Ledger Roadmap

## MVP (V1)
- Manual entry (Expenses/Income/Transfers)
- Currency segregation (EUR/CNY)
- Multi-profiles (Personal/Business)
- Basic CSV imports (Revolut, Wise, Poste Italiane)
- Notification listener for WeChat/Alipay
- Ultra-fast entry mode (Widgets/Shortcuts)

## V2: Auto Bank Sync
- GoCardless Bank Account Data integration (Revolut, Poste Italiane, Wise)
- Lightweight backend (Supabase / Cloudflare Workers)

## V3: Reporting & Intelligence
- Monthly/category pie charts, trend lines
- Real-time ECB exchange rate sync
- Budget tracking and rules engine enhancements
- Cloud sync with custom accounts

## V4: Expansion
- Receipt OCR scanning
- AI-based auto logic for transaction categorization
- Multi-user collaborative ledger
