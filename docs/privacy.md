# 隐私政策 / Privacy Policy

**最后更新：2026年2月**

## 数据收集

本应用（记账本）**不会收集、上传或分享任何用户数据**。所有数据完全保存在用户设备本地。

## 通知监听权限

当用户选择开启"微信/支付宝通知监听"功能时，应用会读取微信和支付宝的推送通知以自动记账。

- 所有通知数据**仅在设备本地处理**
- **不会上传**到任何服务器
- 用户可以随时在设置中关闭此功能
- 关闭后将立即停止读取任何通知

## 存储权限

应用需要存储权限用于 CSV 文件的导入和导出功能。导入的文件数据仅保存在本地数据库中。

## 联系方式

如有隐私相关问题，请通过 GitHub Issues 联系我们。

# Privacy Policy (English)

**Last updated: February 2026**

## Data Collection

This app (Ledger) does **not** collect, upload, or share any user data. All data is stored entirely on the user's device.

## Notification Listener Permission

When the user enables the "WeChat/Alipay notification listener" feature, the app reads push notifications from WeChat and Alipay for automatic expense tracking.

- All notification data is **processed locally on the device only**
- Data is **never uploaded** to any server
- Users can disable this feature at any time in Settings
- Disabling immediately stops all notification reading

## Storage Permission

The app requires storage permission for CSV import and export functionality. Imported file data is stored only in the local database.

## Contact

For privacy-related questions, please contact us via GitHub Issues.
